# 商品管理システム

## 概要
商品の登録・編集・削除を行うアプリケーションです。

## 主な機能
- 商品一覧表示
- 商品登録機能
- 商品一括登録機能
- 商品編集機能
- 商品削除機能

## 開発環境
- PHP 8.1.12
- Laravel 8.83.3
- MySQL
